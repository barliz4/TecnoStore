# 🛒 TecnoStore - Tienda Tecnológica

Bienvenido a **TechStore**, una tienda web moderna enfocada en la venta de productos tecnológicos con una interfaz atractiva, dinámica y responsive.

🔗 **Ver Demo en Vivo:** [TecnoStore](https://barliz4.github.io/TecnoStore/) 

---

## 🚀 Descripción

Este proyecto consiste en el desarrollo de una página web para una tienda tecnológica, diseñada para ofrecer una experiencia de usuario fluida, moderna y visualmente impactante.

El sitio permite visualizar productos, conocer sus características y simular una experiencia de compra dentro de un entorno atractivo tipo e-commerce.

---

## 🎯 Objetivos del Proyecto

- Crear una interfaz moderna y llamativa  
- Implementar diseño responsive (adaptable a móviles, tablets y PC)  
- Simular un catálogo de productos tecnológicos  
- Mejorar la experiencia del usuario (UX/UI)  
- Aplicar buenas prácticas de desarrollo web  

---

## 🛠️ Tecnologías Utilizadas

- 🌐 HTML5  
- 🎨 CSS3 (Flexbox, Grid, Animaciones, Variables CSS)  
- ⚡ JavaScript  
- 💻 Visual Studio Code  

---

## 📱 Características

- ✅ Diseño completamente responsive  
- ✅ Navbar moderno con efectos visuales  
- ✅ Catálogo de productos dinámico  
- ✅ Animaciones suaves y efectos glow  
- ✅ Interfaz inspirada en tiendas tecnológicas profesionales
